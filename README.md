## Road to Next App

A modern **Next.js application** built to explore scalable architecture, clean UI design, and production-ready patterns using **TypeScript, Tailwind CSS, and shadcn/ui**.

---

## Overview

**Road to Next App** is a learning-driven project focused on mastering:

- Next.js App Router
- Modern UI with Tailwind CSS + shadcn/ui
- Clean architecture & separation of concerns
- Type safety with TypeScript
---

## Features

- Dynamic ticket system
- Status-based UI (OPEN, IN_PROGRESS, DONE)
- Reusable UI components (shadcn)
- Dynamic routing for ticket details
- Fast and responsive design

---

## Tech Stack

- **Framework:** Next.js (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS (v4)
- **UI Components:** shadcn/ui
- **Utilities:** clsx
